# Food Delivery Data Analysis - Hackathon 2026

An end-to-end data analysis pipeline that merges SQL, JSON, and CSV datasets to uncover consumer behavior insights.

## Features
- **Multi-source Integration**: Combines relational SQL data, semi-structured JSON, and CSV.
- **Revenue Analytics**: Identifies high-value cities and cuisines.
- **Membership Segmenting**: Analyzes the impact of 'Gold' membership on order value.

## How to Run
1. Install pandas: `pip install pandas`
2. Run the script: `python analysis.py`
